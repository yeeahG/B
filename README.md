# 👭B
B는 Be me, Be you 내가 되고 너가 된다는 뜻을 가지고 있습니다. 글을 통해 생각을 얘기하고 기록하는 소셜 네트워크 서비스이자 마이크로 블로그 서비스입니다.  가볍고 심플한 소셜 네트워크 블로그가 있었으면 좋겠다는 생각에서 계획하게 되었습니다. 텍스트와 사진으로 글을 작성하고 다른 사용자의 글도 볼 수 있으며
